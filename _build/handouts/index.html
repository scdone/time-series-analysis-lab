Multivariate Time Series Analysis

Introduction

In this lab, you will apply what you have learned from lectures on linear regression and mining time series data to build a multivariate dynamic linear regression model (ARIMAX) and create from it scenario based forecasts.

This is an advanced topic and takes the material from earlier lectures (interpreting data using stats, probability and distributions, and linear regression) and a lab (regression mini project) and applies those concepts to a niche of regression analysis: time series data. You may need to review the lecture material from Mining Data from Time Series to help you understand this type of regression analysis.
